	 make crawler


To execute the file, run:

	 ./crawler 

with the crawler configuration file

To compile the search, use the command: 

	make 
	or
	make search 

To execute the file, run: 

	./search

with the search configuration file

<br><br>

To clean and rerun, run:

	make clean
